## 技術的知見（累積・要点のみ）
1. Jules は `a19156664-bot` でPR作成、`google-labs-jules[bot]` はコメント担当
2. `night_date_now()` は JST-12h の日付。`NIGHTLY_FORCE_DATE` で上書き可（turn-switch.yml の input にもあり）
3. Windows PowerShell 5.1 は cp932 → `PYTHONIOENCODING=utf-8` 必須
4. `statusCheckRollup` は Actions トークン不可 → `with_checks=False` 運用、CI列は `no-checks` 表示（正常）
5. tasks.yml の prompt はファイル経由（`.nightly/prompts/*.md` + prompt_file）
6. タスクIDは毎夜 T1-01 / T2-01 リセット（`T[12]-\d{2}` のみ有効）
7. ローカルで実行可能な nightly.py サブコマンドは `validate` のみ（他は gh 依存、Actions 上で実行）
8. **branch protection は API 直接書き込み（gh api PUT contents）も拒否する**（★本日の新知見。朝レポート409の原因）
9. 「Run workflow」=最新コードで新規実行 / 「Re-run jobs」=古いコードで再実行（★本日の新知見）
10. tasks.yml の実スキーマは night / turn1 / turn2 + 各タスクの id/title/risk/paths/prompt_file + test_paths/protected_paths（★本日の新知見）
11. Jules は .egg-info を作ることがある → レビューで除去
12. Claude Code の自動承認済み: git *, python3 -c *, validate系, git update-index *, git fetch *（git commit は都度承認運用）
